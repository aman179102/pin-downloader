# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by opening a **private GitHub issue** at:

https://github.com/aman179102/pin-downloader/issues

Do **not** post security vulnerabilities publicly on GitHub Discussions, public issues, or social media.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Affected versions

## Responsible Disclosure

We will acknowledge your report within 5 business days and work with you to address the issue. We kindly ask that you allow us reasonable time to fix the issue before disclosing it publicly.

## No Committed Secrets

This repository does not contain any API keys, passwords, certificates, or other sensitive credentials in its source code or commit history. If you find any such data, report it immediately via the process above.
